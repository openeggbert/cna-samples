Catapult Wars

This is a C++ / CNA port of the Microsoft XNA Game Studio 4.0 Catapult Wars Lab sample (originally a Windows Phone game). Two catapults take turns lobbing boulders at each other across a windy valley; the first to five hits wins. The human player faces a simple AI opponent.

Note: the original XNA training kit ships its documentation as a Word document rather than the usual SampleName.htm; this file was authored for the CNA port to describe the desktop controls and to drive the F1 help overlay. See missing.md.

Sample Controls

The XNA original is driven entirely by touch; the CNA desktop port maps that to the mouse and keyboard.

Action Control
Aim and fire your catapult (on your turn) Press and drag the mouse, then release
Stronger shot Drag farther before releasing
Select a menu entry Up / Down arrows then Enter, or click the entry
Pause / back out of a menu Escape
Dismiss the end-game screen Click or press Enter
Show / hide this help (10 s) F1
Exit Escape at the main menu
