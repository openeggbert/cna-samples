Honeycomb Rush Sample

Honeycomb Rush is a single-player game in which the player collects honey from a set of honeycombs while avoiding the bees flying around the screen. If the player collects enough honey and returns it to a honey vat within the level's time limit, the game advances to the next (harder) level. The game is built on the Game State Management sample's ScreenManager framework.

Sample Overview

The game includes a main menu, instructions screen, the main gameplay screen, a pause screen, and a high-score screen. The player controls the BeeKeeper, who can move around the screen and fire smoke to repel bees. Worker bees wander randomly; soldier bees chase the BeeKeeper. Beehives regenerate honey over time and are the BeeKeeper's source of honey; depositing honey in the vat before it overflows the level's timer advances to the next difficulty (Easy → Medium → Hard). Reaching the end of Hard, or running out of time, ends the game and offers a high-score entry.
